# Visualizing Women in Science

This repository contains the source code for the project [Visualizing Women in Science](https://womeninscience.amphilsoc.org), created by the [Center for Digital Scholarship](https://www.amphilsoc.org/center-digital-scholarship) at the American Philosophical Society.

## Project requirements:

- [Jekyll >=4.3.1](https://jekyllrb.com/)
- [D3.js >= 7](https://d3js.org/)
- [jQuery >= 3.2.1](https://jquery.com/)

## Contributing

To contribute intellectual content, please open an issue or reach out to the CDS via email.

*Visualizing Women in Science* is not currently under active development. To contribute code, please open an issue first outlining your proposed changes and their rationale.

## Installation instructions

1. Install [Ruby](https://www.ruby-lang.org/en/downloads/) and [Jekyll](https://jekyllrb.com/docs/).
2. In your terminal, navigate to where you want to install the site, then clone this repo and move into the directory the process creates:

```
git clone git@github.com:AmericanPhilosophicalSociety/visualizing-women-in-science.git
cd visualizing-women-in-science
```

3. Install the Jekyll dependencies

```
bundle install
```

4. Run the site locally:

```
bundle exec jekyll serve
```

5. Navigate to http://localhost:4000/ to preview the site.
